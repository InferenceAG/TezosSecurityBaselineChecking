# Testcase title: Michelson opcodes VOTING_POWER and TOTAL_VOTING_POWER

## Description

This scenario checks the correct functioning of the Michelson opcodes VOTING_POWER and TOTAL_VOTING_POWER. It also checks whether companion and consensus key do not have any voting power.

### Assumption
This test case assumes that there is a baker running with the in the test configured companion and consensus keys.

### Expected results

Both Michelson opcode work as expected.
